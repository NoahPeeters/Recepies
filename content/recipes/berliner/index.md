Fluffige Berliner (auch Krapfen oder Pfannkuchen genannt) mit Marmeladenfüllung - ein Klassiker zu Karneval und Silvester.

{{< recipeinfo >}}

{{< ingredients servings="14" >}}
{{< portiontype name="Berliner" plural="Berliner" multiplier="1" default="true" >}}
{{< ingredient-section "Teig" >}}
{{< ingredient ref="1140100.milch" amount="125" id="milch" >}}
{{< ingredient ref="1531200.hefe" amount="42" >}}
{{< ingredient ref="2040202.mehl" amount="500" >}}
{{< ingredient ref="2130100.zucker" amount="100" id="zucker-teig" >}}
{{< ingredient ref="2120100.prise-salz" constant="1" >}}
{{< ingredient ref="1180100.ei" amount="2" >}}
{{< ingredient ref="1140700.butter" amount="100" note="weich" >}}

{{< ingredient ref="2110100.frittierfett" amount="1000" note="Sonnenblumen- oder Rapsöl" >}}
{{< ingredient ref="2130100.zucker" amount="100" id="zucker-waelzen" >}}
{{< ingredient-section "Füllung" >}}
{{< ingredient ref="2020200.marmelade" amount="280" note="z.B. Himbeer oder Erdbeere" >}}
{{< /ingredients >}}

{{< steps >}}

{{< step >}}
{{< use id="milch" >}} in einem kleinen Topf lauwarm erwärmen. {{< use id="1531200.hefe" >}} hineinbröckeln und unter Rühren auflösen.
{{< /step >}}

{{< step >}}
{{< use id="2040202.mehl" >}}, {{< use id="zucker-teig" >}}, {{< use id="2120100.prise-salz" >}}, {{< use id="1180100.ei" >}} und {{< use id="1140700.butter" >}} in eine Schüssel geben. Die Hefemilch dazugeben und mit den Knethaken des Handrührers ca. 5 Minuten zu einem glatten, elastischen Teig verkneten.
{{< /step >}}

{{< step >}}
Den Teig zugedeckt an einem warmen Ort ca. 45 Minuten gehen lassen, bis er sich verdoppelt hat.
{{< /step >}}

{{< step >}}
Den Teig auf einer bemehlten Arbeitsfläche nochmals durchkneten und ca. 2 cm dick ausrollen. Mit einem runden Ausstecher (ca. 8 cm) 14 Kreise ausstechen.
{{< /step >}}

{{< step >}}
Die Teiglinge auf ein mit Backpapier belegtes Blech legen und zugedeckt nochmals 30 Minuten gehen lassen.
{{< /step >}}

{{< step >}}
{{< use id="2110100.frittierfett" >}} in einem weiten Topf auf 170-180 °C erhitzen. Die richtige Temperatur ist erreicht, wenn sich an einem Holzstäbchen kleine Bläschen bilden.
{{< /step >}}

{{< step image="frittieren.webp" >}}
Die Berliner portionsweise ins heiße Fett geben und bei geschlossenem Deckel ca. 3-4 Minuten goldbraun backen. Wenden und weitere 3-4 Minuten ohne Deckel fertig backen.
{{< /step >}}

{{< step >}}
Mit einem Schaumlöffel herausnehmen und auf Küchenpapier abtropfen lassen. Noch warm in {{< use id="zucker-waelzen" >}} wälzen.
{{< /step >}}

{{< step >}}
{{< use id="2020200.marmelade" >}} in einen Spritzbeutel mit Fülltülle füllen. Von der Seite in jeden Berliner ca. 20 g Marmelade spritzen.
{{< /step >}}

{{< /steps >}}

## Tipps

- Der helle Ring entsteht durch das Backen mit Deckel - die Oberseite bleibt dabei hell
- Öl nicht zu heiß werden lassen, sonst werden die Berliner außen dunkel und innen roh
- Fertige Berliner am besten am gleichen Tag genießen
