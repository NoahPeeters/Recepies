Knusprige Kartoffelpuffer nach Omas Rezept - außen goldbraun und knusprig, innen weich und aromatisch.

{{< ingredients servings="4" >}}
{{< portiontype name="Portion" plural="Portionen" multiplier="1" default="true" >}}
{{< ingredient amount="1000" unit="g" name="Kartoffel" plural="Kartoffeln" note="festkochend" >}}
{{< ingredient amount="1" name="Zwiebel" plural="Zwiebeln" >}}
{{< ingredient amount="2" name="Ei" plural="Eier" >}}
{{< ingredient amount="30" unit="g" name="Mehl" >}}
{{< ingredient constant="1" name="TL Salz" plural="TL Salz" >}}
{{< ingredient constant="1" name="Prise Muskatnuss" plural="Prisen Muskatnuss" >}}
{{< ingredient amount="150" unit="ml" name="Pflanzenöl" note="zum Braten" >}}
{{< /ingredients >}}

{{< steps >}}
{{< step >}}
{{< use id="kartoffel" >}} schälen und auf einer Reibe fein reiben.
{{< /step >}}
{{< step >}}
Die geriebenen Kartoffeln in ein sauberes Küchentuch geben und kräftig auspressen, um überschüssige Flüssigkeit zu entfernen.
{{< /step >}}
{{< step >}}
{{< use id="zwiebel" >}} schälen und ebenfalls fein reiben.
{{< /step >}}
{{< step >}}
Kartoffeln und Zwiebeln in einer großen Schüssel vermischen. {{< use id="ei" >}}, {{< use id="mehl" >}}, {{< use id="tl-salz" >}} und {{< use id="prise-muskatnuss" >}} hinzufügen und gut vermengen.
{{< /step >}}
{{< step >}}
{{< use id="pflanzenöl" m="0.5" >}} in einer großen Pfanne erhitzen.
{{< /step >}}
{{< step >}}
Je 2 EL Kartoffelmasse in die Pfanne geben und flach drücken. Bei mittlerer Hitze 3-4 Minuten pro Seite goldbraun braten.
{{< /step >}}
{{< step >}}
Auf Küchenpapier abtropfen lassen und warm halten, während die restlichen Puffer gebraten werden.
{{< /step >}}
{{< /steps >}}

## Tipps

- **Knuspriger:** Die Kartoffeln so trocken wie möglich auspressen.
- **Servieren mit:** Apfelmus, Räucherlachs und Crème fraîche, oder Zucker und Zimt.
- **Warmhalten:** Im Backofen bei 80°C warmhalten.
