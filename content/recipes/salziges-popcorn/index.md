Letzings Geheimtrick - so gelingt salziges Popcorn wirklich wie im Kino! Der Trick ist das gleiche Prinzip wie beim süßen Popcorn: Öl sehr heiß erhitzen, Salz einrühren und dann sofort den Herd ausstellen.

{{< recipeinfo >}}

{{< ingredients servings="2" >}}
{{< ingredient ref="2040100.popcornmais" amount="50" id="mais" >}}
{{< ingredient ref="2110100.oel" amount="60" id="oel" note="4 EL" >}}
{{< ingredient ref="2120100.salz" amount="6" id="salz" note="1 TL" >}}
{{< /ingredients >}}

{{< steps >}}

{{< step >}}
{{< use id="oel" >}} in einem großen Topf erhitzen. Der Topf muss groß sein, da aus 50 g Mais sehr viel Popcorn wird. Das Öl muss richtig heiß werden!
{{< /step >}}

{{< step >}}
{{< use id="salz" >}} einrühren und etwa 10 Sekunden rühren. Dann sofort den Herd ausstellen - das ist sehr wichtig! {{< use id="mais" >}} in den Topf schütten, kurz einrühren (Achtung: die Körner könnten schon anfangen zu fliegen) und Deckel drauf.
{{< /step >}}

{{< step >}}
Zwischendurch den Topf schütteln, damit alles mit dem Salz-Öl-Gemisch bedeckt wird. Nach einigen Minuten ist das Popcorn fertig.
{{< /step >}}

{{< step >}}
Das Popcorn schnell in eine andere Schüssel füllen und lüften lassen. Das Lüften sorgt dafür, dass Flüssigkeit verdampft. Ohne Lüften wird das Popcorn matschig!
{{< /step >}}

{{< /steps >}}

## Tipps

- Verwende unbedingt Popcornmais (Puffmais). Er hat eine dünne Schale und platzt besonders gut auf.
- Popcorn immer ohne Deckel auskühlen lassen, damit es knusprig bleibt.
- Wenn das Poppgeräusch nachlässt (2-3 Sekunden zwischen den Pops), ist das Popcorn fertig.
